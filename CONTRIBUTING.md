# Contributing

## Development

Clone project:

```bash
$ git clone git@github.com:ns3777k/mongoose-migrate.git
```

Start local mongodb:
 
```bash
$ docker run --rm -p27017:27017 mongo
```

To run javascript use `babel-node`: 

```bash
$ ./node_modules/.bin/babel-node src/cli.js
```
